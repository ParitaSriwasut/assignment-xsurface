# Backend

## Setup

### Install docker

Please follow the steps here to install docker: https://docs.docker.com/engine/install/ 2.

### Install node

```
brew install node
```

### This project uses pnpm

```
npm install -g pnpm
```

## Running the code

### Install dependencies

```
pnpm i
```

### Setup Database for local

```
pnpm db:setup
```

### Run the server

```
pnpm dev
```
